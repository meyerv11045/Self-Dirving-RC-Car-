## Self-Driving RC-Car
#### Objective: 
1. Lane Guidance  
2. Stop Sign Detection 

#### System Design: 
1. **Input Unit:** Pi Camera Module V2

2. **Processing Unit:** Raspberry Pi
  
3. **Motor Controller:** Arduino with L293n Motor Driver
